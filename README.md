# $Based claimer

Script to claim $based tokens

## How to start?
- Install Python
- git clone https://github.com/iurii2002/based_claimer.git
- pip install -r requiremets.txt
- create file /files/keys/
- add keys to file /files/keys/
- run use_claimer.py
